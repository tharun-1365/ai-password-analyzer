# Project: AI Password Pattern & Weakness Detection System

## Objective

Build a system that analyzes passwords, detects patterns, classifies strength using ML, and provides improvement suggestions.

## Rules

* Do NOT store user passwords
* Focus on security and explainability
* Use simple and clear code (student-friendly)

## Features

* Password strength classification (Weak/Medium/Strong)
* Pattern detection (sequential, repeated, common passwords)
* Feedback generation
* Optional entropy calculation

## Tech Stack

* Python (Flask/Streamlit)
* scikit-learn for ML

## Output Requirements

* Show strength level
* Show reasons for weakness
* Provide suggestions to improve password

## Goal

Make the project suitable for academic submission and viva demonstration.
